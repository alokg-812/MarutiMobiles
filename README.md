# 📱 MarutiMobiles

**MarutiMobiles** is a full-stack web application for a **mobile phone repair service with home pickup and delivery**, built using **Python Flask** and **server-side rendered templates**. The platform allows customers to book phone repair requests online and track their repair status, while providing an admin dashboard to manage repair workflows efficiently.

### 🚀 Features

* Customer registration & secure login
* Phone repair booking with pickup scheduling
* Repair status tracking (Requested → Picked Up → Under Repair → Ready → Delivered)
* Customer repair history & profile management
* Admin dashboard for managing repair requests
* Role-based access control (Customer / Admin)
* Clean, responsive UI using Bootstrap (Soft Tech Mint theme)

### 🛠 Tech Stack

* **Backend:** Python, Flask
* **Authentication:** Flask-Login / Flask-Security
* **Database:** SQLite (via Flask-SQLAlchemy)
* **Frontend:** Jinja2, HTML5, CSS3, Bootstrap
* **Charts (Admin):** Chart.js (UI-only)

### 🔒 Architecture Highlights

* Server-side rendering (no frontend frameworks)
* Programmatic database creation
* Clean Flask project structure
* Secure role-based access control
* Mobile-first responsive design

### 🎯 Use Case

Designed for local mobile repair businesses to digitize repair bookings, improve service tracking, and offer a smooth customer experience without complex frontend frameworks.
